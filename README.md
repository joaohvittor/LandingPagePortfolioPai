# 🏗 Site Institucional – Serviços de Construção e Caça Vazamentos

Este projeto consiste no desenvolvimento de um site institucional para um prestador de serviços da construção civil, com foco em presença digital e captação de novos clientes.

O objetivo é apresentar de forma clara os serviços oferecidos e facilitar o contato direto com potenciais clientes.

---

## 🎯 Objetivo do Projeto

Criar uma presença online profissional para um prestador de serviços autônomo, permitindo:

- maior credibilidade no mercado
- fácil acesso às informações de serviço
- contato rápido via WhatsApp
- apresentação visual de trabalhos realizados

---

## 🧰 Tecnologias Utilizadas

- HTML5  
- CSS3  
- Bootstrap  

---

## 💼 Aplicação

Este site foi desenvolvido para um profissional da área de:

- reformas residenciais
- caça vazamentos
- instalação hidráulica
- serviços gerais de construção

Com o objetivo de:

- aumentar visibilidade
- gerar novos orçamentos
- melhorar a confiança do cliente

---

## 🚀 Funcionalidades

- Apresentação profissional dos serviços
- Seção institucional (sobre o profissional)
- Galeria de trabalhos realizados
- Botão de contato direto via WhatsApp
- Layout responsivo (adaptado para celular)

---

## 📱 Foco do Projeto

O site foi pensado para usuários que acessam principalmente via celular, facilitando o contato imediato e a solicitação de orçamento.

---

## 🔄 Status

O projeto está em processo de melhoria contínua, com futuras atualizações de layout e experiência do usuário.
